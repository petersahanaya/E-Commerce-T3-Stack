### 💁🏼‍♂️ Shoes Application - P3Commerce
P3Commerce is a shoes application where you can choose and buy an shoe that you want, built using Next Js, tRPC and other technology.

### 💡 Features
* Authentication with Oauth
* Cart system 
* Payment checkout 
* tRPC Api
